# הפעלת Pro3duct

## פתיחת היישום

1. לחץ פעמיים על `Pro3duct.bat`.
2. יופיע מסך פתיחה שמציג מה המערכת טוענת.
3. בהפעלה הראשונה התהליך עשוי לקחת מספר דקות.
4. כשהכול מוכן, ייפתח חלון Pro3duct עצמאי ללא שורת כתובת או טאבים.

אין צורך לפתוח דפדפן ואין צורך להירשם.

## סגירת היישום

- אפשר לסגור את חלון Pro3duct כמו כל תוכנה רגילה.
- כדי לעצור גם את שירותי הרקע, לחץ פעמיים על `stop_pro3duct.bat`.

## במקרה שהיישום לא נפתח

1. ודא ש־Docker Desktop מותקן.
2. פתח את Docker Desktop והמתן עד שהוא מוכן.
3. לחץ שוב על `Pro3duct.bat`.
